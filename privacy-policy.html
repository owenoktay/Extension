Privacy Policy for Background Color Changer Extension

Last updated: 9/5/2024

1. Data Collection

The Background Color Changer extension does not collect, store, or transmit any personal information or user data.

2. Extension Functionality

This extension only changes the background color of web pages when explicitly requested by the user. It does not interact with any personal information or browsing data.

3. Third-Party Access

There is no third-party access to any data, as no data is collected or stored by this extension.

4. Changes to This Policy

If any changes are made to this privacy policy, they will be reflected on this page.
